# ⚽ Premier Predictor Web Dashboard

Este es el landing page (portada) para el modelo de predicción de la Premier League desarrollado por **Allison y Danna**.

## 🎨 Diseño y Estructura
Inspirado en la estética de [Snacklins](https://snacklins.com/), este front-page cuenta con:
- **Video Background**: Fondo dinámico con el video corporativo.
- **Navegación Dinámica**: Menús desplegables que muestran los componentes realizados en el taller (Modelos, Contexto, Fichas).
- **Selector de Idioma**: Un botón ovalado semi-transparente para alternar entre Español e Inglés.
- **Glassmorphism**: Estilos modernos con transparencias y desenfoque (backdrop-filter) para un acabado premium.

## 🚀 Cómo ejecutar localmente
Para ver el proyecto en tu navegador:
1. Abre una terminal en la carpeta `ml-dashboard`.
2. Ejecuta `npm install` (si tienes Node.js instalado).
3. Ejecuta `npm run dev`.
4. Abre la URL que aparezca (normalmente `http://localhost:5173`).

Alternativamente, puedes abrir el archivo `index.html` directamente en tu navegador, aunque algunas funcionalidades de video o assets locales pueden verse restringidas por políticas de seguridad del navegador (CORS). Se recomienda usar un servidor local (Live Server o Vite).
